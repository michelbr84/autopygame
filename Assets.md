# Asset List for Tetris-Style Game

## Images

- assets/images/background.png: 800x600 pixels, used as the game background.
- assets/images/block_I.png: 32x32 pixels, represents the I-piece.
- assets/images/block_O.png: 32x32 pixels, represents the O-piece.
- assets/images/block_T.png: 32x32 pixels, represents the T-piece.
- assets/images/block_S.png: 32x32 pixels, represents the S-piece.
- assets/images/block_Z.png: 32x32 pixels, represents the Z-piece.
- assets/images/block_J.png: 32x32 pixels, represents the J-piece.
- assets/images/block_L.png: 32x32 pixels, represents the L-piece.
- assets/images/buttons/start.png: 100x50 pixels, start button image.
- assets/images/buttons/pause.png: 100x50 pixels, pause button image.
- assets/images/buttons/restart.png: 100x50 pixels, restart button image.

## Sounds

- assets/sounds/rotate.wav: WAV format, played when a tetromino rotates.
- assets/sounds/drop.wav: WAV format, played when a tetromino drops.
- assets/sounds/clear.wav: WAV format, played when lines are cleared.
- assets/sounds/gameover.wav: WAV format, played when the game ends.

*All image files are PNG format and all sound files are WAV format. Ensure the files are placed in the specified directories relative to the project root.*
